# Maxence

## Boring Money

An interactive generator of unsexy, unglamorous, deeply profitable businesses you can actually start.

**Live site:** [boring-money.html](boring-money.html)

### Features
- 30 boring businesses with real margins, startup costs, and step-by-step plans
- Filtering by industry, budget, team size + shuffle
- Pro Pack ($19) with cold email templates, ROI calculators, and startup checklists
- Brutalist newspaper aesthetic
- Single HTML file, zero dependencies
